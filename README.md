# Chronometer

Elegant and intuitive web-based timer.

## Installation

```bash
# clone the repo
git clone https://github.com/pablodorrio/chronometer
```

## Usage

```bash
# change the working directory to chronometer
cd chronometer

# start the server on an available port
serve -l 3000
```
